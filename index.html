Flashcard

JavaScript is a high-level, dynamic, interpreted programming language primarily used for enhancing the interactivity of web pages. It was originally developed by Netscape as a means to add interactive and dynamic elements to websites.

Operators are symbols or keywords that perform operations on one or more operands. Operators allow you to manipulate data, perform calculations, compare values, and control the flow of your program.

Variables are named storage location in computer memory that holds data or values. Variables are fundamental to programming languages like JavaScript because they allow developers to store, manipulate, and retrieve data during the execution of a program.

Variable naming conventions refer to guidelines and rules that developers follow when naming variables in programming languages like JavaScript. Consistent and meaningful variable names enhance code readability, maintainability, and understanding.

A function is a named block of code that performs a specific task or operation. Functions are fundamental building blocks in most programming languages, including JavaScript. Functions are essential for structuring and organizing code in JavaScript and other programming languages.

A method is a kind of function that is called on something.A method is a function that is associated with an object. Methods are defined as properties of objects and are used to perform actions or provide functionality related to that object.

Arguments refer to the values passed to a function when it is called or invoked. These values are received by the function through its parameters, which act as placeholders for the incoming arguments. Arguments allow functions to perform tasks with different inputs dynamically.

Parameters are placeholders defined in the function declaration to receive values (arguments) when the function is called or invoked. Parameters allow functions to accept inputs dynamically and perform tasks based on those inputs.

Return is a keyword used within functions to specify the value that the function should output or "return" when it is called and executed. It allows functions to compute a result and send that result back to where the function was called from.

Chaining methods refers to the technique of calling multiple methods on an object in a single, fluent statement. This approach is facilitated by methods that return the object itself (this) after performing their operation, allowing subsequent methods to be called sequentially on the same object. Chaining methods is commonly used in JavaScript to write concise and readable code.

A string is a sequence of characters enclosed within single quotes (') or double quotes ("). Strings are used to represent text and are a fundamental data type in JavaScript, allowing developers to manipulate and work with textual data in various ways.

Booleans are fundamental data type that represent logical values. A boolean can have one of two possible values: true or false. Booleans are often used in conditional statements and logical operations to control the flow of a program based on whether certain conditions are true or false.

Undefined is a primitive value that is automatically assigned to variables that have been declared but have not been assigned a value. It indicates that a variable or object property has not been initialized and does not currently have a value assigned to it. undefined is also returned by functions that do not explicitly return a value.

NaN stands for "Not-a-Number". It is a special value that represents an invalid or unrepresentable value as defined by the IEEE floating-point standard. NaN is returned when a mathematical operation cannot produce a meaningful result.

Null is a special value that represents the intentional absence of any object value. It is a primitive value that indicates a variable or object property does not currently have a value assigned to it.

Alert is a global function that displays a modal dialog box with a specified message and an OK button. It is primarily used for displaying simple notifications or messages to the user, often used in scenarios where immediate attention or confirmation is required.

Comments are annotations within the code that are ignored by the JavaScript interpreter during execution. They are used by developers to add explanations, notes, or documentation directly within the source code. Comments are essential for improving code readability, aiding in understanding, and facilitating collaboration among developers.

jQuery is a fast, small, and feature-rich JavaScript library designed to simplify client-side scripting of HTML.
